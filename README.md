# Grupo-C2A-022019-Front-End

## Development

- Install dependencies

```bash
npm install
```

- Run dev server

```bash
API_URL=http://<your-api-dev-server>/ npm run dev
```

optionally you can omit environment variable API_URL and it will default to `http://localhost:8080/`

- Run tests

```bash
npm run test
```
